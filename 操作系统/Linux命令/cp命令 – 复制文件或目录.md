# cp命令 – 复制文件或目录

参考资料：

[cp命令 – 复制文件或目录 – Linux命令大全(手册) (linuxcool.com)](https://www.linuxcool.com/cp)

---

‍

**语法格式：**cp [参数] 源文件 目标文件

‍

**常用参数：**

|参数|解释|
| ------| ----------------------|
|-r|递归复制文件和目录<br />|

‍

**参考案例：**

```shell
[root@linuxcool ~]# cp -r Documents Doc		# 在当前工作目录中，将某个目录复制一份，并定义新目录名称
```
